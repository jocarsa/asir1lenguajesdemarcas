Cuando trabajamos con Flask:

1.-Tenemos un archivo Python que importa Flask
2.-AL LADO de ese archivo python debéis tener una carpeta llamada "templates"
3.-Dentro de esa carpeta templates, ponéis el/los HTML
4.-El nombre del archivo llamado desde python debe coincidir con el nombre del archivo HTML

Si el archivo se llama 001-blog.html
Desde Python no podéis llamarlo con blog.html

O bien cambiais el nombre al archivo
O bien desde python lo llamáis 001-blog.html
