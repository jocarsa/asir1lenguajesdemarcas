Cada uno de vosotros va a desarrollar un blog personal
Un blog es una web en la cual podéis publicar noticias de vosotros mismos
Modelo de datos: una sola tabla, con: titulo, fecha, autor, contenido, imagen
1.-Debéis crear una base de datos en SQLite y crear una tabla de "posts" o "entradas" o "noticias" (importante copiad todos los comandos que correspondan)
2.-Debéis crear un controlador en Python que se conecte a la base de datos y recupere las noticias
3.-Debéis crear una web (HTML y CSS) y pintar los artículos del blog, en esa web (usando templates)

En el examen, habrán tres preguntas de desarrollo (intercalaréis explicación y código):

1.-Explica paso a paso cómo has hecho la base de datos
2.-Explica paso a paso cómo has hecho el script de Python que se conecta a SQLite, recupera datos, levanta una web, y pinta esos datos (flask)
3.-Explica paso a paso cómo has hecho el HTML y el CSS de la web que has levantado

En el examen se espera que traigáis todo el código desarrollado
